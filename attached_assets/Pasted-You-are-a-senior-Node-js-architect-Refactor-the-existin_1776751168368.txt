## STRICT REQUIREMENTS

### 1. REMOVE EXPRESS COMPLETELY

* Remove all express imports

### 2. SETUP FASTIFY SERVER

Create:

/src/app.ts

* Initialize Fastify instance
* Register plugins

/ src/server.ts

* Start server
* Handle errors

Use:

* fastify()
* fastify.register()
* fastify.listen()

Enable:

* Logger

---

### 3. ROUTE MIGRATION

Convert ALL routes to Fastify style.

Example conversion:

FROM (Express):

```js
app.post('/agent/analyze', controller.analyze);
```

TO (Fastify):

```ts
fastify.post('/agent/analyze', async (request, reply) => {
  return controller.analyze(request, reply);
});
```

---

### 4. CONTROLLER UPDATE

Controllers must:

* Accept (request, reply)
* NOT use (req, res)

Example:

```ts
async analyze(request, reply) {
  const result = await service.run(request.body.projectId);
  return result;
}
```

### 5. VALIDATION (IMPORTANT)
